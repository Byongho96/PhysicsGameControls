[name]

PhysicsControls is a controller responsible for simulating physical phenomena.
It moves objects based on the `velocity` value and simulates gravity, collision detection, friction.

Constructor

[name]( [3param:https://threejs.org/docs/#api/en/core/Object3D object], [param:HTMLDOMElement domElement], [3param:https://threejs.org/docs/#api/en/core/Object3D world] )

[3page:https://threejs.org/docs/#api/en/core/Object3D object] — The `Object3D` to apply physics controls to.
[page:HTMLDOMElement domElement] — The HTML element for event listeners.
[3page:https://threejs.org/docs/#api/en/core/Object3D world] — The `Object3D` used to build the collision octree.

Properties

[property:Collider collider]

The collider used for collision detection. Default is `new Collider()`.

[property:Number gravity]

The gravitational force applied to the object. Default is `30`.

[property:boolean isGrounded]

(Read-only) Whether the object is currently grounded.

[property:boolean isLanding]

(Read-only) Whether the object is currently landing.

[property:Number landTimeThreshold]

The time threshold in seconds for determining if the object is landing. Default is `0.3`.

[property:Number landTolerance]

The distance tolerance for landing detection, measured as the vertical distance from the ground surface. Default is `0.6`.

[property:Number maxFallSpeed]

The maximum fall speed of the object. Default is `20`.

[property:Number maxXBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Number maxYBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Number maxZBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Number minXBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Number minYBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Number minZBoundary]

The boundary limits for the object. Defaults is `Infinity`.

[property:Vector3 resetPoint]

The position to reset the object when it goes out of bounds. Default is `new THREE.Vector3(0, 0, 0)`.

[property:Number resistance]

Resistance applied to the object to dampen velocity. Default is `6`.

[property:Number step]

The time step for calculating physics. Larger values result in more physics calculations performed per unit of time. Default is `5`.

[property:Vector3 velocity]

The velocity vector of the object. Default is `new THREE.Vector3(0, 0, 0)`.

[property:Object3D world]

The current world object used to build the collision octree.

[property:BVH worldBVH]

The BVH structure of the world object used for collision detection.

Methods

Returns the velocity into the object's local coordinate system.

[method:undefined update]( [param:Number delta] )

Update the controls. The `delta` is the time elapsed since the last update (in seconds).

Events

collide

Fires when the collider has collided with the world. The event object includes:

  • type: `'collide'`
  • normal: A `Vector3` representing the collision normal.
