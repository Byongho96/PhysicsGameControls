[page:Object animationClips] — The Animation clips. Supported animations are:
'IDLE' | 'MOVE_FORWARD' | 'RUN_FORWARD' | 'MOVE_BACKWARD' | 'RUN_BACKWARD' | 'MOVE_LEFTWARD' | 'RUN_LEFTWARD' | 'MOVE_RIGHTWARD' | 'RUN_RIGHTWARD' | 'JUMP_UP' | 'LAND' | 'FALL' // Example of animationClips { 'IDLE': idleClip, 'MOVE_FORWARD': moveClip, 'RUN_FORWARD': runClip, ... } [3page:https://threejs.org/docs/#api/en/cameras/Camera camera] — The camera following the character.
