[name]

FirstPersonControls is a control system operated through actions performed by the user. It updates the `velocity` inherited from PhysicsControls based on the actions.

Constructor

[name]( [3param:https://threejs.org/docs/#api/en/cameras/Camera object], [param:HTMLDOMElement domElement], [3param:https://threejs.org/docs/#api/en/core/Object3D world] )

[3page:https://threejs.org/docs/#api/en/cameras/Camera object] — The camera to be controlled.
[page:HTMLDOMElement domElement] — The HTML element for event listeners.
[3page:https://threejs.org/docs/#api/en/core/Object3D world] — The `Object3D` used to build the collision octree.

Properties

[property:Number accelerationFactor]

Multiplier for movement speed when accelerating. Default is `1.5`.

[property:Object actionStates]

Defines the values for each action. The magnitude of each action is proportional to the assigned value. Default is:

{ 'MOVE_FORWARD': 0, 'MOVE_BACKWARD': 0, 'MOVE_LEFTWARD': 0, 'MOVE_RIGHTWARD': 0, 'JUMP': 0, 'ACCELERATE': 0, 'ROTATE_UP': 0, 'ROTATE_DOWN': 0, 'ROTATE_RIGHT': 0, 'ROTATE_LEFT': 0 };

[property:Boolean enableAcceleration]

Whether to enable acceleration when holding the accelerate key. Default is `true`.

[property:Boolean enableZoom]

Whether to enable zooming with the mouse wheel. Default is `true`.

[property:Number eyeHeight]

Height of the camera from the ground. Default is `1.5`.

[property:Number floatingMoveSpeed]

Speed of movement when floating. Default is `8`.

[property:Number groundedMoveSpeed]

Speed of movement when grounded. Default is `30`.

[property:Number jumpForce]

Force applied for jumping. Default is `15`.

[property:Number maxTiltAngle]

Maximum rotation tilt-up angle (radians). Default is `Math.PI / 2`.

[property:Number maxZoom]

Maximum zoom level. Default is `Infinity`.

[property:Number minTiltAngle]

Minimum rotation tilt-down angle (radians). Default is `- Math.PI / 2`.

[property:Number minZoom]

Minimum zoom level. Default is `0`.

[property:Number rotateSpeed]

Speed of rotation. Default is `1`.

[property:Number zoomSpeed]

Speed of zooming. Default is `1`.
