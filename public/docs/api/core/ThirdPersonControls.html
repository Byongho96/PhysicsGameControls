[name]

Executes the appropriate animation based on the object's current state. Additionally, the camera tracks and rotates around the character.

Constructor

[name]( [3param:https://threejs.org/docs/#api/en/core/Object3D object], [param:HTMLDOMElement domElement], [3param:https://threejs.org/docs/#api/en/core/Object3D world], [param:Object animationClips], [3param:https://threejs.org/docs/#api/en/cameras/Camera camera] )

[3page:https://threejs.org/docs/#api/en/core/Object3D object] — The character to be controlled.
[page:HTMLDOMElement domElement] — The HTML element for event listeners.
[3page:https://threejs.org/docs/#api/en/core/Object3D world] — The `Object3D` used to build the collision octree.
[page:Object animationClips] — The Animation clips. Supported animations are:
'IDLE' | 'MOVE_FORWARD' | 'RUN_FORWARD' | 'MOVE_BACKWARD' | 'RUN_BACKWARD' | 'MOVE_LEFTWARD' | 'RUN_LEFTWARD' | 'MOVE_RIGHTWARD' | 'RUN_RIGHTWARD' | 'JUMP_UP' | 'LAND' | 'FALL' // Example of animationClips { 'IDLE': idleClip, 'MOVE_FORWARD': moveClip, 'RUN_FORWARD': runClip, ... } [3page:https://threejs.org/docs/#api/en/cameras/Camera camera] — The camera following the character.

Properties

[property:Object animationClips]

(Read-only). Returns a frozen copy of the animation clips associated with the character.

[3property:https://threejs.org/docs/#api/en/cameras/Camera camera]

The time for transitioning between animations. Default is `0.3`.

[3property:https://threejs.org/docs/#api/en/math/Vector3 cameraLookAtOffset]

Offset for the camera look-at position relative to the object. The default is defined based on the object's size: `new Vector3(0, objectSize.y * 0.8, 0)`.

[3property:https://threejs.org/docs/#api/en/math/Vector3 cameraPositionOffset]

Offset for the camera position relative to the object. The default is defined based on the object's size: `new Vector3( 0, objectSize.y * 2, - objectSize.y * 2 )`.

[property:Number cameraLerpFactor]

Lerp factor for smooth camera transitions. Default is `1`.

[property:Number fallSpeedThreshold]

The speed threshold to trigger the falling animation. Default is `15`.

[property:Number moveSpeedThreshold]

The speed threshold to trigger the moving animation. Default is `1`.

[property:Number runSpeedThreshold]

The speed threshold to trigger the running animations. Default is `8`.

[property:String syncAxisWithCamera]

The value that determines when to synchronize the object's directional coordinates with the camera's directional coordinates. Default is `MOVE`.

    Possible values:
  • `'ALWAYS'`: The object's forward axis is always synchronized with the camera.
  • `'MOVE'`: The object's forward axis is synchronized with the camera only when the object is moving.
  • `'NEVER'`: The object's forward axis is not synchronized with the camera.

[property:Number transitionDelay]

The delay for transitioning between animations. Default is `0.3`.

[property:Number transitionTime]

The time for transitioning between animations. Default is `0.3`.
