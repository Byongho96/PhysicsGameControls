Mixins

Provide Keyboard input for controlling the object.
`keyToActions` is overwritten:

keyToActions = { 'KeyW': [ 'MOVE_FORWARD' ], 'KeyS': [ 'MOVE_BACKWARD' ], 'KeyA': [ 'MOVE_LEFTWARD' ], 'KeyD': [ 'MOVE_RIGHTWARD' ], 'Space': [ 'JUMP' ], 'ShiftLeft': [ 'ACCELERATE' ], 'ArrowUp': [ 'ROTATE_UP' ], 'ArrowDown': [ 'ROTATE_DOWN' ], 'ArrowRight': [ 'ROTATE_RIGHT' ], 'ArrowLeft': [ 'ROTATE_LEFT' ] };
