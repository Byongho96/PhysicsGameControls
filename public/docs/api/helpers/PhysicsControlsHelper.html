[name]

A helper class for visualizing the collider of a `PhysicsControls` instance. This class uses a `CapsuleGeometry` to represent the collider and dynamically updates its position, rotation, and size.

Code Example

const controls = new FPKeyboardDragControls( camera, renderer.domElement, world.scene ); const helper = new PhysicsControlsHelper( controls, 0xffffff ); scene.add( helper );

Constructor

[name]( [param:PhysicsControls controls], [param:Number color] )

[page:PhysicsControls controls] — The camera to be controlled.
[page:PhysicsControls controls] — Hex color code representing the capsule collider.(Ex. `0xff0000`)

Properties

[property:String type]

The type of the object. Always `'PhysicsControlsHelper'`.

Methods

[method:undefined update]()

Updates the position, rotation, and size of the helper to match the collider.

[method:undefined dispose]()

Disposes of the helper's geometry and material to free up resources.
