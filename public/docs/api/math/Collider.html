[name]

Extends the `Capsule` class to represent a 3D capsule-shaped collider. This class includes additional properties and methods for managing the collider's position and length.

Constructor

[name]( [param:Vector3 start], param:Vector3 end], [param:Number radius] )

[3page:https://threejs.org/docs/#api/en/math/Vector3 start] — `Vector3` value for the starting position of the capsule body.
[3page:https://threejs.org/docs/#api/en/math/Vector3 end] — `Vector3` value for the ending position of the capsule body.
[page:Number radius] — Radius of the capsule collider.

Properties

[property:Number length]

Gets or sets the length of the capsule (distance between the start and end points). Setting this property adjusts the start and end points while preserving the center position.
