[name]

A mixin that extends `FirstPersonControls` (which includes `ThirdPersonControls`) to add drag-based rotation capabilities. The mixin allows for customizable drag actions.

Constructor

[name]( [param:FirstPersonControls base] )

[page:FirstPersonControls base] â€” A class that extends `FirstPersonControls`.

Properties

[property:Array dragXActions]

An array of actions to be triggered by horizontal mouse movement. Default is `['ROTATE_RIGHT']`.

[property:Array dragYActions]

An array of actions to be triggered by vertical mouse movement. Default is `['ROTATE_DOWN']`.

[property:Number dragDampingFactor]

The damping factor applied to drag actions when [page:.enableDragDamping] is true. Default is `1`.

[property:Boolean enableDragDamping]

Determines whether drag damping is enabled. Default is `true`.

Methods

[method:void connect]()

Connects the drag functionality by adding event listeners.

[method:void disconnect]()

Disconnects the drag functionality by removing event listeners.

[method:void dispose]()

Disposes of the drag functionality by cleaning up event listeners and resources.

[method:void update]( [param:Number delta] )

Update the mixin. Must be called if [page:.enableDragDamping] is set. The `delta` is the time elapsed since the last update (in seconds).
