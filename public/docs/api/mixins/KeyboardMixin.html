A mixin that extends `FirstPersonControls` (which includes `ThirdPersonControls`) to add keyboard-based action mapping. This mixin allows for customizable key-to-action mappings, enabling precise control through keyboard inputs.

Properties

[property:Object keyToActions]

A mapping of keyboard keys to actions. Each key is associated with one or more actions. Default is:

{ 'KeyW': ['MOVE_FORWARD'], 'KeyS': ['MOVE_BACKWARD'], 'KeyA': ['MOVE_LEFTWARD'], 'KeyD': ['MOVE_RIGHTWARD'], 'Space': ['JUMP'], 'ShiftLeft': ['ACCELERATE'] }
