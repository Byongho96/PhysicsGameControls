[name]

A mixin that extends `FirstPersonControls` (which includes `ThirdPersonControls`) to add pointer lock functionality. This mixin provides a mechanism to track mouse movements and map them to actions, enabling immersive control experiences.

Constructor

[name]( [param:FirstPersonControls base] )

[page:FirstPersonControls base] â€” A class that extends `FirstPersonControls`.

Properties

[property:Array pointerLockXActions]

An array of actions to be triggered by horizontal mouse movements. Default is `['ROTATE_RIGHT']`.

[property:Array pointerLockYActions]

An array of actions to be triggered by vertical mouse movements. Default is `['ROTATE_DOWN']`.

[property:Number pointerLockDampingFactor]

The damping factor applied to pointer lock actions when [page:.enablePointerLockDamping] is true. Default is `1`.

[property:Boolean enablePointerLockDamping]

Determines whether pointer lock damping is enabled. Default is `true`.

Methods

[method:void connect]()

Connects the pointer lock functionality by adding event listeners for `mousedown` and `mousemove`.

[method:void disconnect]()

Disconnects the pointer lock functionality by removing event listeners for `mousedown` and `mousemove`.

[method:void dispose]()

Disposes of the pointer lock functionality by cleaning up event listeners and resources.

[method:void update]( [param:Number delta] )

Update the mixin. Must be called if [page:.enablePointerLockDamping] is set. The `delta` is the time elapsed since the last update (in seconds).
