Three Game Controls provides game controls built on top of [link:https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent/code three.js]. It features essential physics engine capabilities, such as gravity and collision detection, making it lightweight and suitable for use in web environments. It also supports interaction with various input devices, such as mouse, keyboard, and touch

Motivation

With the advancement of WebGL technology, 3D experiences on the web are becoming increasingly common. However, creating realistic virtual worlds remains a cumbersome task. Three Game Controls aims to minimize boilerplate code and provide a simple controller API that allows characters to explore immersive worlds using only the three.js codebase.

Three Game Controls will likely:

  • Lightweight:
    Focuses on core physics features like gravity and collision detection, ensuring high performance in the web environment.
    (If you require more detailed physics capabilities, consider other options.)
  • Customizable:
    Supports a wide range of device I/O combinations to control characters. In addition to the default controls we provide, you can create your own controllers using Mixins.
  • Animation Management:
    The third-person controller automatically plays appropriate animations based on the current velocity. Developers can also customize animation clip sources as needed.

Although the name is "Game Controls", it is not limited to games. Use it to make your services, products, or ideas more captivating in 3D spaces.
